Process Data Program Documentation

This page documents a Python program that reads paired numeric data from a text file and stores it as coordinate tuples.

Program Purpose

This program reads paired numeric data from a text file and stores it as coordinate tuples in a list. The file is expected to contain one data pair per line, formatted as: 

x,y

How the Program Works

  1. Specifies a file path to a text file (e.g., data/data01.txt).
  2. Initializes an empty list named data_list to hold all data pairs.
  3. Opens the file for reading and processes it line by line.
  4. For each line:
    • Removes extra whitespace using strip().
    • Splits the line by the comma using split(',').
    • Converts the values to numbers using float().
    • Appends the pair as a tuple (x, y) into data_list.
  5. Prints a confirmation message and displays the resulting list.

Python Code (Reference)

file_path = 'data/data01.txt'  # Replace with your actual file path

data_list = [] # Initialize an empty list to store the data

with open(file_path, 'r') as file:     # Open and read the file
    for line in file:
        # Strip whitespace and split by comma
        x, y = line.strip().split(',')  
        # or int(x), int(y) if your data is in integer format
        data_list.append((float(x), float(y)))       
print("Data successfully read from file.")
print("Data List:", data_list)

Example Input and Output

Example file contents (data01.txt):

1.2,3.4
5.6,7.8

Resulting list stored in data_list:

[(1.2, 3.4), (5.6, 7.8)]
Note: The program assumes every line has exactly two values separated by a comma. If your file contains integers instead of decimals, you can convert with int(x) and int(y) instead of float().
